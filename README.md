# SimpleBluetoothScanner

A simulated Bluetooth scanning iOS application built using UIKit and MVVM.

## Features
- Simulated Bluetooth scan using Timer
- Devices revealed one per second
- JSON-based device data
- Duplicate prevention
- Programmatic UI with Auto Layout

## Architecture
- MVVM
- Codable for JSON parsing
- UIKit (no Storyboards)

## Note
This project uses simulated Bluetooth data and does not require CoreBluetooth or Bluetooth permissions.
